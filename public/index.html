File Uploader Demo

Simple Rails app that demonstrates multi-part file upload.

About your application’s environment

This is a simple Rails app that demonstrates multi-part file upload. The only change from a barebones Rails app is the addition of the FileUploadController and associated views.

I wrote it to test the FileTransfer function of PhoneGap. Uploaded files are placed in the /tmp directory and available for manual inspection.
